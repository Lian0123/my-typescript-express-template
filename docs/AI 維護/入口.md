# AI 維護入口

本入口是所有 AI 與維護者的唯一第一站。目標是用最少 token 找到可信來源，不要求先讀完整份文件庫。

## 30 秒專案摘要

- Node.js + TypeScript 4.9 + Express 4 後端範本。
- Inversify 負責依賴注入；TypeORM 連接 PostgreSQL；RabbitMQ 傳遞帳號／角色事件。
- Socket.IO 提供 `/v1/chatting` 即時訊息雛形；Redis 與 MongoDB 有連線或模型基礎，但功能尚未完整。
- HTTP 功能集中在帳號與角色；啟動組裝點是 `src/main.ts` 與 `src/container.ts`。
- 正確性以程式碼、設定、測試執行結果為準，文件是可維護索引而非第二份程式碼。
