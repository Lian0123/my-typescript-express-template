# 功能總索引

| 功能／責任 | 入口 | 延伸位置 | 搜尋詞 |
|---|---|---|---|
| 啟動與 middleware | `src/main.ts` | `src/env.ts` | `InversifyExpressServer`, `setConfig` |
| 依賴注入與外部連線 | `src/container.ts` | `src/typeorm-config.ts` | `containerBinding`, `bind<` |
| 帳號與角色 | `src/accounts/` | `src/test/accounts/` | `V1Account`, `V1Role` |
| 時序資料 | `src/timeseries/` | `src/test/timeseries/` | `V1TimeSeries`, `time_series_samples` |
| RabbitMQ 事件 | `src/accounts/*.event.controller.ts` | `src/utils/rabbit-mq.ts` | `createRabbitMQEvent`, `listenRabbitMQEvent` |
| 即時聊天 | `src/chatting/chatting.socket.controller.ts` | `src/chatting/`、`src/redis/` | `ChattingController`, `@onMessage` |
| 錯誤與驗證 | `src/common/constants/error.const.ts` | `src/utils/util.ts` | `ErrorHandler`, `validateClass`, `serviceError` |
| 資料庫 migration | `src/migrations/` | `src/typeorm-migration-config.ts` | `MigrationInterface` |
| Swagger/OpenAPI | controller decorators | `src/swagger/index.html`、`script/shell/create-openapi-generate.sh` | `ApiOperation`, `ApiPath` |
| 測試 | `src/test/` | `jest.config.js` | `.spec.ts`, `.e2e.spec.ts` |
| 容器與環境 | `docker-compose*.yaml` | `docker/dockerfile`、`envs/.env.example` | `process.env`, `${` |

新增獨立功能域時，在本目錄新增 `<功能名>索引.md`，並從本頁與入口連結；索引只保留責任、資料流、主要路徑、測試與風險。
