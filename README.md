# my-typescript-express-template

TypeScript、Express、Inversify、TypeORM、RabbitMQ 與 Socket.IO 的後端服務範本。

給人類與 AI 的專案導航請從 [AI 維護入口](docs/AI%20維護/入口.md) 開始。該文件採漸進式讀取設計，會依任務導向最小必要內容。

```bash
npm run build
npm run lint
npm run test
npm run docs:verify
```

環境變數請由 `envs/.env.example` 建立本機 `envs/.env`；不要提交密鑰或實際環境設定。
